# 🚀 專案啟動指南

## 📋 前置準備

確保已安裝：
- Node.js 18 或以上版本
- npm 或 yarn

## ⚡ 快速啟動（開發模式）

### 方法一：手動啟動

#### 1️⃣ 啟動後端
```powershell
# 在專案根目錄執行
cd server
npm install
npm start
```

後端將運行於：`http://localhost:3000`

#### 2️⃣ 啟動前端
開啟**新的終端視窗**：
```powershell
# 在專案根目錄執行
cd client
npm install
npm run dev
```

前端將運行於：`http://localhost:5173`

#### 3️⃣ 開啟瀏覽器
訪問：`http://localhost:5173`

---

### 方法二：使用 npm scripts（推薦）

在專案根目錄可以執行以下指令（需先安裝 concurrently）：

```powershell
# 安裝 concurrently（全域）
npm install -g concurrently

# 同時啟動前後端
concurrently "cd server && npm start" "cd client && npm run dev"
```

### 檢查後端

應該看到遊戲首頁，狀態顯示「✅ 已連線」

## 🎮 開始遊戲

1. **輸入玩家名稱**
2. **選擇模式**：
   - 🎲 單人抽蛋：自由練習
   - ⚔️ 對戰模式：雙人對決

### 單人模式
- 選擇扭蛋類型（30/50/100元）
- 選擇抽取數量（1/5/10抽）
- 點擊「開始抽蛋」

### 對戰模式
**玩家A（房主）**：
1. 點擊「建立房間」
2. 選擇扭蛋類型與數量
3. 複製房間號給玩家B

**玩家B（加入者）**：
1. 點擊「加入房間」
2. 輸入房間號
3. 等待對戰開始

## ❓ 常見問題

### Q: 前端顯示「連線中...」
**A:** 檢查後端是否正常運行於 port 3000

### Q: Socket.IO 連線失敗
**A:** 確認防火牆沒有阻擋 3000 和 5173 埠

### Q: 圖片無法顯示
**A:** 將圖片檔案放到 `server/public/items/` 目錄

### Q: 抽卡後沒有反應
**A:** 開啟瀏覽器開發者工具（F12）查看 Console 錯誤訊息
